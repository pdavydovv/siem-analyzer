# SIEM Analyzer (Вариант 10)

Проект для анализа событий безопасности (SIEM). 

## 📋 Описание
Система генерирует, сохраняет и анализирует события безопасности. Каждое событие представлено в виде неизменяемого именованного кортежа (`namedtuple`). 

Основные возможности:
- 🔍 **Фильтрация**: отбор событий по уровню критичности (severity).
- 📊 **Аналитика**: группировка событий по типам (используется `collections.Counter`) и извлечение уникальных IP-адресов (используется `set`).
- 📝 **Отчетность**: генерация полного аналитического отчета в формате JSON.
- 💾 **Сериализация**: сохранение и загрузка состояния объекта анализатора.
- 📜 **Логирование**: запись действий программы в файл `app.log` (уровни INFO и DEBUG).
- 🧪 **Тестирование**: полное покрытие юнит-тестами с использованием `pytest` (позитивные и негативные сценарии).
- 📖 **Документация**: автоматическая генерация сайта документации через `MkDocs`.
